Project Objective

The objective of this project is to convert a static HTML website into a dynamic Node.js application using Express.js for backend routing and server-side processing.

Technologies Used

  • HTML5
  • CSS3
  • JavaScript
  • Node.js
  • Express.js

Project Outcome

After conversion, the website can support backend logic, routing, and future database integration.
